Readme for the Autodesk Product Manager

Installation Requirements

Installation of Autodesk Product Manager requires the following:
  • Microsoft® Internet Explorer 5.01 or later
  • Microsoft Installer
  • MDAC (Microsoft Data Access Components)

Supported Operating Systems

  • Microsoft Windows® 98 or later

Known Limitations and Issues

MDAC (Microsoft Data Access Components)
To save Autodesk Product Manager report files in Excel 2000 or Access 2000 format, you must have MDAC 2.5 or later installed on your computer. If you do not have MDAC installed and you try to save a report in XLS or MDB format, the following message is displayed:

"Unable to save the report in the selected format. Please save in CSV format or see ReadMe_CADMgr.htm for information about how to download the latest MDAC (Microsoft Data Access Components)."

MDAC (Microsoft Data Access Components) is included with Windows 2000 and later, Office 2000, and other applications. It is available for download (free) from the Microsoft website (search term: MDAC).

Once it is installed, MDAC is part of the operating system and is available to other applications.

To download and install MDAC:

  1. Download the mdac_typ.exe file.
  2. Accept the license agreement.
  3. Restart your computer after you install MDAC.
Remote Registry Access for Windows 9x or Me
If any of the computers included in your search definition run Windows 9x or Me, you must enable remote registry access. Without remote registry access, you can run Autodesk Product Manager at the local computer only. For more information, see Knowledge Base Article Number Q141460 on the Microsoft website.
The mfc42.dll File for Windows NT 4.0 or 98
On a Windows NT 4.0 or Windows 98 computer with no other products installed, Autodesk Product Manager does not run, because the mfc42.dll file needs to be updated. If you install AutoCAD 2000 or later or any Office 97 product, the DLL file is updated and Autodesk Product Manager runs.
AutoCAD Mechanical 14.5 Not Found
Autodesk Product Manager does not find installations of AutoCAD Mechanical 14.5.
Registry Access on Windows NT 4.0 and Windows 2000
Access to the registry of Windows NT 4.0 and Windows 2000 computers is not permitted if the following key exists: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\SecurePipeServers\winreg. For more information, see Knowledge Base Article Number Q153183 on the Microsoft website.
